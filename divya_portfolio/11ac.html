Cisco deliverable Feature

• Understanding the Atheros AP Platform architecture.
• Understanding the 80211 interference patterns
• Understanding the 80211 interference detection algorithms.
• Developed feature that allows the Host to set the TX chain mask parameters on a per client basis.
• Developed features that allows the Host SW to get the max regulatory TX power, max and min chip power and max/min power for all rates from the FW
• Developed features that allows the Host SW to set the channel power limits control tables in addition to the ones stored elsewhere (e.g. EEPROM, OTP).
• Optimizing the beacon interval for bursted and staggered beacons.
• Debugging and bug fixing issues in AP software.
